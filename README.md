## Project Overview
The **Hospital Appointment System** is designed to modernize and automate the patient-doctor scheduling process. It provides a secure, real-time platform where patients can book consultations and healthcare providers can manage their daily schedules efficiently.

### Tech Stack
- **Frontend:** [Next.js](https://nextjs.org/) (React Framework)
- **Backend:** [Node.js](https://nodejs.org/)
- **Database & Auth:** [Supabase](https://supabase.com/) (PostgreSQL)
- **Styling:** Tailwind CSS

---

## Key Features
- **Secure Authentication:** Multi-role login (Patient/Doctor) powered by Supabase Auth.
- **Appointment Management:** Real-time booking, rescheduling, and cancellation.
- **Patient Records:** Secure access to medical history and upcoming visits.
- **Doctor Dashboard:** Dedicated view for managing patient lists and daily availability.
- **Responsive Design:** Optimized for mobile, tablet, and desktop browsers.

## Installation & Setup
**1.Clone the repository:**

```
   git clone https://github.com/workwithmanny/Hospital_Appointment_System_Final_Year_Project.git
   cd Hospital_Appointment_System_Final_Year_Project
```

2. **Install dependencies:**

   ```
   npm install
   ```

3. **Configure Environment Variables:**
   Create a `.env.local` file in the root directory and add your Supabase credentials:

   ```
   NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

4. **Run the Development Server:**

   ```
   npm run dev
   ```

   Open [http://localhost:3000](http://localhost:3000) to view the application.
